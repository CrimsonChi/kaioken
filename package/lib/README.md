# **Kaioken**

#### _A lightweight (less than 5Kb) react-like library for creating web apps with ease._

<br />

## Features

- [x] useState
- [x] useEffect
- [x] useContext
- [x] useReducer
- [x] Router
- [x] StyleScope
- [x] useRef
- [x] useMemo
- [x] useFetch
- [x] useSyncExternalStore
- [x] useCallback
- [x] memo
- [x] useOptimistic
